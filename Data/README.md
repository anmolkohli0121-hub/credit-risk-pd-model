# Data Description

This project uses the German Credit dataset, a publicly available dataset
commonly used for benchmarking credit risk models.

Due to repository size considerations and to mirror real-world data
governance practices, the raw dataset is not included in this repository.

## Notes

- The dataset contains borrower demographic, financial, and loan-related attributes.
- No personal or confidential data is included.
- This project uses the dataset for educational and demonstration purposes only.
